Funcionalidades:
    - Ver la lista de productos asignados a un almacén a partir de dar clic en un almacén "x".
    - Añadir stock manualmente a un producto dentro de un almacén.
    - Eliminar stock manualmente de un producto dentro de un almacén (se crea un "Product Exit").
    - Transferir producto de un almacén a otro (se crea un "Product Transfer").
    - Eliminar un producto de un almacén si su stock es 0.
    - ?? Acceder a "Product Summary" desde inventory.

La estructura del código se ordena según Clean Architecture de la siguiente forma:

- data
    - datasources
    - models
    - repositories
- domain
    - entities
- presentation
    - blocs
    - pages
    - widgets
