Funcionalidades:
    - Se accede desde "Catalog"
    - Guardar en local la lista de productos a adquirir.
    - Mostrar un botón para eliminar un producto del carrito.
    - Mostrar un botón para convertir el carrito en una orden de compra.
    - Al convertir a orden de compra se borra toda la información actual del carrito.

La estructura del código se ordena según Clean Architecture de la siguiente forma:

- data
    - datasources
    - models
    - repositories
- domain
    - entities
- presentation
    - blocs
    - pages
    - widgets
